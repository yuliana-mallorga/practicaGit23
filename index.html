comandos git

Prepara los archivos

git add .

Para confirmar y guardar los cambios se pone git commit

git commit -m "mensaje sobre el commit"

Crear y saltar a una nueva rama

git checkout -b "nombreRama"

saltar a otra rama

git checkout -b "nombreRama"

Subir nuestro codigo al repositorio remoto

git push origin nombre de la rama

para lista el historial del commit

git log
